O que é o html?

O HTML é uma linguagem de Marcação de Hipertexto ou seja, é a linguagem que dara conteudo e permitira construir a estrutura visual e organizacional de uma pagina na internet.

Ferramentas utilizadas

Para fazer uso das funçoes do html e darmos os primeiros passos para criação de uma pagina necessitamos apenas de um editor de texto e um Browser de navegação.

Em nosso caso optamos pelo Visual Code para escrever nosso código dowload.E no caso do navegador vamos utilizar o Chrome devido as suas funcionaliades

Inspetor de Elementos

É a ferramenta que podemos utilziar para visualizar o codigo fonte da pagina. Com ela vamos verificando como a pagina vai ficando tambem muito util para usar em conjunto com o CSS e a linguagem utilizada.

Estrutura basica

Podemos dividar o html em duas estruturas basicas.

o head ou cabeçalho onde serão gravadas as informçoes antes do codigo rodas, nelas sao gravadas por exemplo titulos e link do CSS.

E o body ou corpo onde esta toda estrutura da pagina com testos, listas , inputs, links e etc.

Abaixo temos como é a estrutura basica de uma pagina conforme o html.

tags

O html é todo definido pelas tags, ja no inicio delimitamos o documento com a tag html.Elas informam ao nevagor o que eles devem fazer.

No html temos varias operações e as tags determinam o que fazer

temos tags de textos ou topograficas, tags de listas, de inputs, botoes entre muitas outras

Atributos das tags

As tags necessitam de um identificador para acessarmos suas propriedades.

Fazemos isso atraves dos atributos, assim podemos adicionar informçoes as paginas deixando mais trabalhadas, alem de manipular as informaçoes do modo que desejamos,as mais comuns sao os id e as class.

Texto

São as tags utilizada para organizar o texto.

Nela definimos o tamanho dos texto seus paragrafos, e tipos especifios dos mesmos.

Listas

No html temo dois tipo de lista as ordenadas com a tag ol e as não ordenadas com a tag ul.

As listas são muito utilizadas para formação de menu de paginas e funões .

Links

Temos a tag de link a que vai cliar ancora para acessar outras paginas ou menus.

É uma tag muito utilizar para navegar sobre as paginas deixando a estrutura do projeto bem mais elaborada.
